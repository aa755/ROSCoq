Instructions for running the Coq program on an iCreate robot

  1. Follow the steps 1-2 listed for testing the ROS and robot setup
  2. In another terminal, start our shim. In the process of installing the ROSJava and the Shim, our script should have created the script ~/rosjava/runShim.sh The first line of this script set the options for coqtop. The second line indicates the Coq .v file containing the message handler. You should not need to change these environment variables if you followed the installation instructions exactly. Run this script:

    abhishek@abhishek-desktop:~$ cd rosjava
    abhishek@abhishek-desktop:~$ bash runShim.sh

    Wait for 10-20 seconds while Coq finishes loading all files.
  3. Similarly, Start the hardware agent in a new terminal:

    abhishek@abhishek-desktop:~$ cd rosjava
    abhishek@abhishek-desktop:~$ bash runDriver.sh

  4. Finally, start the external agent in another terminal

    abhishek@abhishek-desktop:~$ cd rosjava
    abhishek@abhishek-desktop:~$ bash runExternalAgent.sh
