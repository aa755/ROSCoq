Instructions for installing ROSJava

  1. Download our script. This script can be used to install ROS and iRobot Create drivers.

    abhishek@abhishek-desktop:~$ wget www.cs.cornell.edu/~aa755/ROSCoq/rosjavaInstall.sh

  2. Now run the above script. The script uses apt-get will prompt for permission to download packages. It will create the directory ~/rosjava and install rosjava there. Some of our scripts depend on the this particular choice of directory.

    abhishek@abhishek-desktop:~$ bash -x rosjavaInstall.sh

Creating and compiling the ROSJava package for the ROSCoq Shim

  1. Download our script. This script creates a package with the standard ROSJava example, and replaces those java files with the shim files:

    abhishek@abhishek-desktop:~$ wget www.cs.cornell.edu/~aa755/ROSCoq/mkROSCoqShimROSJavaPackage.sh

  2. Now run the above script:

    abhishek@abhishek-desktop:~$ bash -x mkROSCoqShimROSJavaPackage.sh
